# 🌸 Florescer

Bem-vindo ao **Florescer**, um projeto de front-end desenvolvido para a disciplina de Desenvolvimento Web. O objetivo do projeto foi criar um site para uma floricultura utilizando apenas **HTML5** e **CSS3**, aplicando os principais conceitos estudados em sala de aula.

---

## 📖 Sobre o Projeto

O **Florescer** é um site responsivo que apresenta uma floricultura fictícia, permitindo que os visitantes conheçam os produtos, a história da empresa e realizem encomendas por meio de um formulário.

O projeto foi desenvolvido seguindo boas práticas de organização do código e utilizando HTML semântico para melhorar a estrutura da aplicação.

---

## 🚀 Funcionalidades

- 🏠 Página Inicial
- 🌹 Catálogo de Produtos
- 🌿 Página Sobre
- 📝 Formulário de Encomendas
- 📱 Layout Responsivo
- 🌸 Favicon personalizado
